This behavior for Printer uses Print.js instead of jQuery.print.

To enable this behavior, in the faces-config.xml, in the <behavior>...</behavior> section, add:

<behavior>
    <behavior-id>org.primefaces.component.PrinterBehavior</behavior-id>
    <behavior-class>org.primefaces.extensions.behavior.printer.PrinterBehavior</behavior-class>
</behavior>
